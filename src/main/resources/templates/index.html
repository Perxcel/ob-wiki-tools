Confluence Tools

This is a simple application to retrieve a Open Banking Standards Page Content, and convert it to Markdown.

Page Id:

Page Id is the unique content id for a Confluence Page.

e.g. Page id for International Payments standard is 1077806166

https://openbanking.atlassian.net/wiki/spaces/DZ/pages/1077806166/International+Payments+v3.1.2
